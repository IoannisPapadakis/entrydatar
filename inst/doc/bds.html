General information on the dataset

  • Downloads data from the Census Business Dynamics Statistics: BDS website.
  • The two main data resources are the firm characteristics tables and the establishment characteristics tables
  • Codebook is available here

Using the package

For example to start and get the BED industry data:

library(entrydatar)
dt_firm  <- get_bds_cut(1977, 2014, "firm", "all") %>% data.table
dt_firm[]

Or for another cut, say establishments from age sic (see below for the exact code for each cut)

dt_estab <- get_bds_cut(1977, 2014, "establishment", "agesic") %>% data.table
dt_estab
